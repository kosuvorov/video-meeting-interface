# Video Meeting Interface

A beautiful, simple, and customizable static video meeting interface built with React, Vite, and Tailwind CSS. 
It features a realistic meeting interface with a customizable countdown timer, making it perfect for holding screens or simple countdown experiences.

## Features

- **Countdown Timer**: Automatically starts a countdown before an event begins.
- **Image Upload**: Upload a static image to present as a slide.
- **Presenter Controls**: Configure the event name, duration, and control the timer.
- **Static App**: Fully client-side application with no backend dependencies required.

## Local Development

**Prerequisites:** Node.js

1. Install dependencies:
   ```bash
   npm install
   ```
2. Start the development server:
   ```bash
   npm run dev
   ```

## Deployment

This app is configured to deploy automatically to GitHub Pages using GitHub Actions when pushed to the `main` or `master` branches of your repository.
